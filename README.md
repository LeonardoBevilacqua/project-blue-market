# Project Blue Market

A project to replace the use of Notion tables to track grocery and pet store shopping 

## Current approach

In the current approach, we use Notion pages along with a database. Details in [`current-approach.md`](./docs/current-approach.md)

## New approach

Use an app to manage shopping and improve the existing flow. Details in [`new-approach.md`](./docs/new-approach.md)

## Tech

This project will use:
- `Penpot`, to create the design and prototype.
- `Supabase`, with a free project, to handle data storage and authentication.
- `Kotlin`, to have a good, offline-first mobile experience.
- `SvelteKit`, with static build for free hosting, to have a web app version with more features, like dashboards and charts.
