# Data structure

The ID fields should either UUID or Integer.

- Kind of shopping
    - ID
    - Name: String
- Item type
    - ID
    - Kind ID
    - Name: String
- Item location
    - ID
    - Name: String
- Market list place
    - ID
    - Name: String
- Market list
    - ID
    - Name: String
    - Date: Date
    - Kind ID
    - Place ID
- Market list item
    - ID
    - Name: String
    - Quantity: Floating value
    - Unit value: Floating value
    - Total: Floating value
    - Barcode: integer
    - Type ID
    - Location ID
